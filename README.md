## Re-implementation for Rencos

### Dependency
```cmd
python == 3.6
pytorch == 1.9.0
nltk == 3.6.3
rouge == 1.0.1 ref: https://github.com/pltrdy/rouge
docopt == 0.6.2
```

### Quick Start 
- Build Vocab
```cmd
cd scripts
vocab.bat
```

- Train Model
```cmd
cd scripts
train.bat
```

- Infer
```cmd
cd scripts
infer.bat
```

- Evaluation
```cmd
cd scripts
eval.bat
```
